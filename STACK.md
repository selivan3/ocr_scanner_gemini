# OCR Scanner - Tech Stack

## Язык программирования
- **Python 3.14** — основной язык

## Backend Framework
- **Flask 3.1** — веб-фреймворк

## AI/ML
- **Google Gemini 3 Pro** (`gemini-3-pro-preview`) — облачный OCR и детекция углов

## Computer Vision
- **OpenCV 4.12** — классическая обработка изображений
- **Pillow 12.1** — работа с изображениями

## Структура проекта
- **Clean Architecture** — слоистая архитектура
- **Dependency Injection** — через Container
